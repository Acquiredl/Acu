# SMPTE ST 2110

**Concern:** Carriage, synchronization, and description of video, audio, and ancillary essence streams over IP networks for real-time professional production and broadcast.
**Issuing body:** SMPTE.
**Scope:** Multipart suite — ST 2110-10 (system, clock, RTP), ST 2110-20 (uncompressed video), ST 2110-30 (audio), ST 2110-40 (ancillary data).

## When to cite

Cite when a pipeline touches real-time professional audio/video production or broadcast infrastructure. Pick a specific part (e.g., ST 2110-20 for uncompressed video colour space). Distinct from cinema distribution (that's DCP, ST 428/429).

## Notes

Supersedes legacy SDI baseband standards. Full specs via SMPTE Standards Store (paid); overview public. Not for file preservation (that's separate) — this is a live-production standard.
