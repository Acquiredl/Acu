# NIST DLMF

**Concern:** Authoritative reference for 600+ special functions — notation, definitions, properties, computational methods.
**Issuing body:** NIST.
**Scope:** 36 chapters (3 methodology + 33 function classes); ~10,000 verified equations with ~2,500 citations.

## When to cite

Cite when a pipeline computes or references special functions (Bessel, gamma, hypergeometric, elliptic, …). Cite the specific equation number (e.g., "DLMF 5.11.3").

## Notes

De-facto academic and computational standard; successor to Abramowitz & Stegun. Free and open. Covers reference material, not algorithm implementations.
