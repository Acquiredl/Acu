# MathML 4.0

**Concern:** XML-based markup for encoding mathematical expressions — both visual presentation and computational semantics.
**Issuing body:** W3C.
**Scope:** Two orthogonal vocabularies — Presentation MathML (~38 tags for visual layout) and Content MathML (~170 tags for semantics).

## When to cite

Cite when a pipeline produces or consumes mathematical content for the web, accessibility tools, or computation. Pick presentation vs content vocabulary explicitly. Cite the specific chapter (e.g., "MathML 4.0 Chapter 3 — Presentation").

## Notes

Related: MathML Core is a simplified profile intended for browser engines. Full MathML is used in publishing and accessibility toolchains.
