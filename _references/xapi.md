# xAPI 2.0

**Concern:** Recording and exchanging learning experiences across courses, simulations, on-the-job performance, and mobile contexts.
**Issuing body:** IEEE (formal standardization) / ADL Initiative (community governance).
**Scope:** JSON statement schema (Actor–Verb–Object), REST API, and Learning Record Store (LRS) conformance requirements.

## When to cite

Cite when a pipeline records learner behaviour beyond the LMS — field training, simulation, social learning, workflow performance. Cite the statement structure and any relevant profile (e.g., cmi5).

## Notes

Supersedes/extends SCORM (which tracks only course completion inside an LMS). First open-source learning-technology spec to achieve IEEE standardization. Complements IMS Caliper Analytics (1EdTech) at the event-schema layer.
