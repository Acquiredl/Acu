# CODATA Recommended Values

**Concern:** Internationally-endorsed values (with uncertainty bounds) for ~400 fundamental physical constants.
**Issuing body:** CODATA Task Group on Fundamental Constants.
**Scope:** Least-squares-adjusted values — Planck constant, elementary charge, Avogadro constant, fine-structure constant, and more. Published every ~4 years in Reviews of Modern Physics.

## When to cite

Cite when a pipeline uses a fundamental physical constant in calculation or comparison. Use the CODATA 2022 value with stated uncertainty; justify use of superseded editions explicitly.

## Notes

Referenced by the BIPM SI brochure and ISO metrology documents. Freely accessible at physics.nist.gov.
