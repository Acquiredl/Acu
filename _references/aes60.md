# AES60 — Core Audio Metadata

**Concern:** Creation, management, and preservation of audio material across production, distribution, and archival environments.
**Issuing body:** Audio Engineering Society.
**Scope:** Core metadata schema extending Dublin Core with audio-specific fields (Version, Publication History, Metadata Provider, Entity Type); supports OAI-PMH exchange for archive interoperability.

## When to cite

Cite when a pipeline produces, catalogues, or preserves audio assets (music, broadcast archives, oral history, sound design). Pair with Dublin Core at the resource-metadata layer.

## Notes

Full specification is paywalled via AES; overview and scope are public. Adopted by audiovisual archives and broadcast facilities. Domain-specific extension rather than a replacement for Dublin Core.
