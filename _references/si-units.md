# SI Units

**Concern:** Traceable, coherent measurement units for all scientific work.
**Issuing body:** BIPM.
**Scope:** Seven base units (second, metre, kilogram, ampere, kelvin, mole, candela) defined in terms of seven fixed physical constants; derived units constructed coherently from base units.

## When to cite

Cite when a pipeline emits or compares measured quantities. Cite the specific unit (e.g., "meters per second per SI Brochure §2.3") rather than just "SI".

## Notes

Current SI Brochure is the 9th edition (2019 redefinition: kilogram decoupled from the IPK artefact; Planck constant fixed). De-jure global standard via the Metre Convention.
