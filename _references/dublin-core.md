# Dublin Core

**Concern:** Minimal, cross-domain vocabulary for describing any resource for discovery and interoperability.
**Issuing body:** Dublin Core Metadata Initiative (DCMI).
**Scope:** 15 core elements — Creator, Title, Subject, Description, Publisher, Contributor, Date, Type, Format, Identifier, Source, Language, Relation, Coverage, Rights.

## When to cite

Cite when a pipeline emits, catalogues, or exchanges any described resource — especially when cross-system discovery matters (library catalogues, digital archives, generic metadata). Prefer Dublin Core as the interoperable baseline; layer CIDOC CRM, EAD, or domain-specific schemas on top where depth is needed.

## Notes

Elements are optional, repeatable, and language-agnostic. Qualified Dublin Core adds ~50 element refinements and encoding schemes. De-facto foundation for library and cultural heritage cataloguing globally.
